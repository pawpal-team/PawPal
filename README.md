# 🐾 PawPal | 寵物即時照護平台

「深夜突發不慌張，PawPal讓醫療成為最安心的擁抱。」

An all-in-one pet care platform built for modern families.<br />

### 🔎 ABOUT PLATFORM

**PawPal**是一個以 **「即時醫療導航」** 為核心，並延伸至 **「日常健康護照」** 的一站式全方位毛孩守護平台。

當深夜或緊急危機來臨時，PawPal 能在第一時間為你**整合並篩選出距離最近、營業中且評價優良的動物醫院資訊**，化解飼主無助的慌亂瞬間。

同時，平台進一步延伸出毛孩的**專屬健康護照**，協助飼主結構化整理**就醫紀錄、日常狀況與成長歷程圖表**。PawPal 讓你從**緊急突發的即時救助，到日常生活的細心陪伴**，都能擁有最安心、最溫暖的數位照護體驗。

---

### ✨ FEATURES

#### 🏥 即時醫療導航 (Medical Navigator)

- **突發狀況首選：** 專為手足無措的當下設計，一鍵精準篩選距離最近、評價優良且「真正營業中」的動物醫院，防止緊急載送時現場撲空。

#### 🩺 結構化就醫紀錄 (Medical Logs)

- **醫病溝通橋樑：** 提供病歷表與就醫紀錄功能，協助飼主條理分明地向獸醫說明病況，減少憑記憶回想的溝通誤差。

#### 📊 一站式健康護照 (Pet Passport)

- **全方位數據追蹤：** 集中紀錄每日食量、飲水與排泄狀況，並將數據可視化，透過動態折線圖與柱狀圖長期追蹤體重，精準掌握發育趨勢。

#### 📅 儀表板提醒日程 (Pet Dashboard Calendar)

- **重要行程不漏接：** 整合於寵物日常儀表板中，支援自由增加疫苗接種、驅蟲計畫或回診預約等提醒事項，免疫因忙碌而遺忘的健康風險。
